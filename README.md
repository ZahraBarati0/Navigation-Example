# React E-Commerce SPA

A clean and responsive **React single-page application (SPA)** demonstrating:

- Product listing and detail pages
- Protected checkout route with login/logout
- Reusable UI components: Navbar, cards, buttons
- Responsive design for desktop and mobile
- Built with React Router and plain CSS (no external UI frameworks)

---

## 🚀 Features

- **Protected Routes:** Checkout page accessible only to authenticated users
- **Simple Authentication:** Login/Logout state handled with React hooks
- **Reusable Components:** Easy-to-extend Navbar, buttons, and cards
- **Responsive Layout:** Works seamlessly on desktop and mobile devices

---

## 💻 Getting Started

### Prerequisites

- Node.js >= 14
- npm or yarn

### Installation

```bash
git clone <your-repo-url>
cd <your-repo-folder>
npm install
npm run dev
```
